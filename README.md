# Docker-Compose Express Boilerplate

Dockerised starter example express app. Includes optional Makefile commands.

## Start

```sh
make up
open http://`docker-machine ip dev`:3000
```

## Logs

```sh
make status
```

## Test

```sh
make cli
npm run test
```
